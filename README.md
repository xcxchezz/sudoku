# Игра Судоку

## Описание

Приложение представляет собой классическую игру Судоку с графическим интерфейсом, реализованную на Java с использованием Swing.

### Основные возможности:

- **Генерация головоломок**: Автоматическая генерация корректных головоломок судоку с использованием алгоритма возврата (backtracking)
- **Интерактивный интерфейс**: Удобное графическое поле 9x9 с визуальным разделением на блоки 3x3
- **Проверка решений**: Автоматическая проверка правильности введённых значений
- **Визуальная обратная связь**: 
  - Изначально заполненные ячейки выделены серым цветом и недоступны для редактирования
  - Правильные значения отображаются синим цветом
  - Неправильные значения отображаются красным цветом

### Архитектура проекта:

- **Модель** (`model`):
  - `SudokuBoard` - хранит игровое поле и решение
  - `SudokuGenerator` - генерирует новые головоломки
  - `SudokuSolver` - решает судоку с использованием алгоритма возврата

- **Контроллер** (`controller`):
  - `SudokuController` - управляет логикой игры и проверкой ходов

- **Представление** (`view`):
  - `SudokuFrame` - графический интерфейс пользователя

## Запуск программы

Для запуска программы выполните:

```bash
javac -d out -sourcepath src src/ru/vsu/ru.vsu.sudoku/Main.java
java -cp out ru.vsu.ru.vsu.sudoku.Main
```

Или используйте вашу IDE (IntelliJ IDEA, Eclipse и т.д.) для компиляции и запуска.
